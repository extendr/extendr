Expand description

Computes the Cholesky factorization with Bunch-Kaufman pivoting of the input matrix and stores the factorization in matrix and subdiag.

The diagonal of the block diagonal matrix is stored on the diagonal of matrix, while the subdiagonal elements of the blocks are stored in subdiag.

ยงPanics

Panics if the input matrix is not square.

This can also panic if the provided memory in stack is insufficient (see cholesky_in_place_req).
